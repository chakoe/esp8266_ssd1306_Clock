## 🚀 部署指南

### 1. 硬件准备
- ESP8266开发板 (建议NodeMCU)
- SSD1306 128x64 OLED显示屏
- DS1307 RTC模块 + 备用电池
- 4个按键开关 + 10kΩ上拉电阻
- 面包板和连接线

### 2. 软件烧录
```bash
# 使用Arduino CLI
arduino-cli compile --fqbn esp8266:esp8266:generic esp8266_ssd1306_Clock.ino
arduino-cli upload --fqbn esp8266:esp8266:generic --port /dev/ttyUSB0 esp8266_ssd1306_Clock.ino

# 或使用一键部署脚本
bash deploy_production.sh
```

### 3. 首次配置
1. 设备启动后长按K4键进入配网模式
2. 连接WiFi热点 "Clock_Device_2.0"
3. 在浏览器中配置网络参数
4. 设备自动重启并连接网络
