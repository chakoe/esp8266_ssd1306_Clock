# ESP8266 SSD1306 智能时钟项目

## 项目概述

这是一个基于ESP8266的智能时钟项目，具备网络时间同步、RTC备份、OLED显示、按键控制等多种功能。项目设计为生产级应用，具有高稳定性和可靠性。

## 主要特性

- **多时间源支持**: 支持NTP网络时间同步、DS1307 RTC硬件时钟、手动设置等多种时间源
- **智能时间源切换**: 当NTP不可用时自动切换到RTC，RTC失效时使用软件时钟
- **OLED显示**: 使用SSD1306 128x64 OLED显示屏，支持亮度调节和自动调暗
- **WiFi连接**: 使用WiFiManager实现便捷的WiFi配置和自动重连
- **按键控制**: 支持4个按键，实现时间设置、亮度调节、显示模式切换等功能
- **电源管理**: 包括自动调暗、睡眠模式、低功耗管理
- **安全特性**: WiFi密码AES加密存储、硬件看门狗保护
- **I2C通信管理**: 健壮的I2C通信错误处理和恢复机制
- **生产级部署**: 包含完整的生产环境部署脚本和监控系统

## 硬件要求

- ESP8266开发板（推荐NodeMCU或D1 Mini）
- SSD1306 OLED显示屏（128x64分辨率，I2C接口）
- DS1307 RTC模块
- 4个按键开关
- 面包板和连接线

## 软件依赖

- Arduino IDE 或 Arduino CLI
- 以下Arduino库：
  - U8g2
  - NTPClient
  - RTClib
  - WiFiManager
  - Ticker

## 功能说明

### 时间管理
- **NTP时间同步**: 通过pool.ntp.org获取网络时间，每小时自动同步
- **RTC备份**: 使用DS1307作为硬件时钟备份，断电后仍可保持时间
- **时间源智能切换**: 根据可用性自动切换时间源（NTP > RTC > 手动）

### 显示管理
- **时间显示**: 以大字体显示当前时间（时:分:秒）
- **日期显示**: 显示当前日期和星期
- **状态信息**: 显示WiFi连接状态、时间源信息等
- **亮度调节**: 支持4级亮度调节，包含自动调暗和夜间模式
- **显示设置**: 可切换大小字体显示

### 按键功能
- **K1键**: 
  - 在设置模式，短按: 增加键
- **K2键**: 
  - 在设置模式，短按: 减小键
  - 短按:切换大小字体
  - 长按：进入时间源设置后，短按：切换时间源，短按：K4键确认保存退出设置
- **K3键**: 
  - 短按：切换亮度
  - 长按：进入时间设置模式，短按切换选择：日期或时间，短按：k1键(增加)或k2键(减小)
- **K4键**: 
  - 长按: 进入配网模式
  - 短按：显示或隐藏网络状态
  - 设置模式中: 短按：确认保存退出

### 电源管理
- **自动调暗**: 5分钟后自动降低屏幕亮度
- **睡眠模式**: 30分钟后进入低功耗睡眠模式
- **夜间模式**: 22:00-7:00自动降低亮度

### 安全特性
- **WiFi密码加密**: 使用AES-128加密存储WiFi密码
- **硬件看门狗**: 8秒超时的硬件看门狗保护
- **错误恢复**: 完善的错误检测和自动恢复机制

## 部署说明

### 1. 环境准备
确保已安装Arduino CLI或Arduino IDE，并安装所需的库。

### 2. 烧录固件
```bash
# 使用Arduino CLI烧录
arduino-cli compile --fqbn esp8266:esp8266:d1_mini esp8266_ssd1306_Clock.ino
arduino-cli upload --fqbn esp8266:esp8266:d1_mini --port COM3 esp8266_ssd1306_Clock.ino
```

### 3. 首次配置
1. 设备启动后，如果检测到K4长按或首次启动，会进入配网模式
2. 连接WiFi热点 "Clock_Device_[VERSION]"
3. 在浏览器中打开配置页面（通常是192.168.4.1）
4. 输入WiFi凭据并保存
5. 设备将自动重启并连接到指定网络

### 4. 验证部署
- 检查显示屏是否正常显示时间
- 验证WiFi连接状态（右上角WiFi图标）
- 确认时间同步功能（时间应每秒更新）
- 测试按键操作功能

## 配置选项

### WiFi配置
- SSID: 在配网界面设置
- 密码: AES-128加密存储
- 自动重连: 启用
- 连接超时: 20秒

### 时间设置
- NTP服务器: pool.ntp.org
- 时区: UTC+8 (北京时间)
- 自动同步: 每小时一次
- RTC备用: 启用

### 显示设置
- 刷新率: 1Hz
- 亮度级别: 4档 (0-3)
- 自动调暗: 5分钟后
- 夜间模式: 22:00-7:00

## 故障排除

### 常见问题
1. **无法连接WiFi**: 
   - 检查SSID和密码是否正确
   - 确认WiFi信号强度
   - 重启设备进入配网模式重新配置

2. **时间不更新**: 
   - 检查网络连接状态
   - 验证NTP服务器可达性
   - 检查RTC模块连接

3. **显示异常**: 
   - 检查I2C连接（SDA、SCL线）
   - 确认OLED供电正常
   - 检查I2C地址是否正确（默认0x3C）

4. **按键无响应**: 
   - 检查按键接线和上拉电阻
   - 确认按键引脚定义正确

### 日志调试
在生产模式下，调试日志默认禁用。如需启用调试，可在config.h中定义DEBUG_MODE。
